Employee & Project Dashboard
Overview

A lightweight SPA for managing employees, projects, and assignments with real-time financial calculations and vacation tracking.

Features
Employee & Project CRUD
Assignment management (capacity, fit)
Financial calculations (revenue, cost, profit)
Sorting, filtering, and table views
Vacation calendar affecting availability
Data persistence via LocalStorage (monthly snapshots)
🛠 Tech Stack
HTML, CSS, Vanilla JavaScript
LocalStorage
Run the Project
git clone <your-repo-url>
cd employee-dashboard

Open index.html in your browser
(or use Live Server in VS Code)
 Notes
Data is saved automatically
Each month is stored separately
Seed data is available for quick start
